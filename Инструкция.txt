1. Скачать репозиторий приложения:
https://github.com/AnatolyTimakov/SecureVault
Разархивируйте и поместите в удобное место папку с проектом.

2. Установка окружения Node.js 
Перейти на официальный сайт: https://nodejs.org/
Скачать installer и установить через него Node.js (везде выбираем стандартные настройки)

3. Открыть консоль, перейти через команду cd в папку с проектом:
Пример: cd \Documents\SecureVault

4. Запустите команду npm install

5. После установки пакетов для проекта запускаем локальный сервер командой: npm start

6. В консоли введите: http://localhost:3000/pwsafe.html
